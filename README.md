# PropertyDemo

Vue 3 + Java 21 demo project.

## Folder Layout

```text
PropertyDemo/
  frontend/   Vue 3 + Vite
  backend/    Java 21 + Spring Boot 3
```

## Quick Preview

If you only want to see a page immediately, open:

```text
frontend/standalone.html
```

This page can be opened directly in a browser.

## Frontend

Requirements:

- Node.js 20 LTS or newer

Run:

```bash
cd frontend
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```

## Backend

Requirements:

- JDK 21
- Maven 3.9+

Run:

```bash
cd backend
mvn spring-boot:run
```

Open:

```text
http://localhost:8080/api/hello
```
